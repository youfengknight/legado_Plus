# 更新日志


## cronet版本: 128.0.6613.40
 
**2026/05/2**
- 新增URL记录功能，支持记录用户访问的URL，包括书源、订阅、搜索等
- 数据库版本升级为93
- 增加字重模式设置，支持精细模式和粗略模式，默认粗略模式。安卓9以上版本支持精细模式。
- 优化在编辑内容保存后光标不会自动跟随的问题


**2026/05/1**
- 新增切换规则功能，支持在代码编辑界面切换不同规则
- 修复代码编辑界面切换规则后，代码内容不刷新的问题
- 翻页速度设置，范围50-1000ms，默认300ms。手动点击和音量点击都可设置
- 优化帮助文档无限Dialog问题
- 增加查询规则功能，支持在代码编辑界面快速查询当前源所有规则并快速跳转编辑
- 增加切换源规则功能，支持在代码编辑界面切换当前源的不同规则
- 页眉页脚字体大小调节，范围12-24px，默认12px。
- 去除滚动条

**2026/04/30**
- 修复部分文档查看问题
- 优化文档选择器界面，移除重复的"帮助"标签
- 优化发现跳动问题
- 修复源编辑页面滚动问题
- 去除书架界面左侧滚动条
- 关闭每次只保留最新备份功能，防止新版空数据备份覆盖旧版数据，引起数据丢失问题

**2026/04/29**
- 修复书源管理中选择书源添加分组时不显示已创建分组的bug
- 优化书架分组滑动流畅度
- 封面设置补全缺失的中文翻译
- 优化帮助文档选择器界面，移除重复的"帮助"标签
- 新增阅读记录功能，支持四种显示模式：汇总视图、时间线视图、最后阅读、阅读时长
- 阅读记录支持热力图日历展示，可按阅读次数或阅读时长显示
- 阅读记录支持滑动删除、合并同名书籍功能
- 阅读记录兼容旧版数据，自动修复作者信息
- 阅读记录UI适配主题背景和颜色

**2026/04/28**
- 优化Room升级bug
- 书源编辑/订阅编辑增加JSON编辑功能
- 源编辑界面增加滚动条
- 订阅管理增加排序功能
- 编辑内容菜单增加搜索
- 导出书单增加导出为json功能
- 主订阅页面同步排序设置，与订阅源管理页面保持一致
- 书源编辑界面修复内容过长导致界面滚动问题，滚动条固定长度

**2026/04/27**
- 修复Web服务中文乱码问题
- 发现页面右上角增加全部添加书架功能，自动检测是否已添加书架
- 帮助文档编辑内容bug修复
- 新增波浪线下划线模式
- 内置Webview增加查看cookie
- 发现和订阅界面增加滚动条
- 其他设置界面增加搜索
- 书签搜索功能补全
- 增加帮助文档全局搜索


**2026/04/26**
- 修复备份功能未包含主题背景图片的问题
- 修复恢复备份后主题背景图片不显示的问题

**2026/04/25**
- 主题列表新增批量操作功能，长按主题进入多选模式
- 支持批量删除、批量导出主题
- 支持批量导入主题（JSON数组格式）
- 增加日志和崩溃日志、阅读记录搜索功能
- 增加复制崩溃日志到剪贴板功能
- 增加帮助文档选择器，支持快速选择各种教程等帮助文档

- 主题和排版可以批量删除

**2026/04/24**
